Sequências

Uma sequência é uma lista ordenada de termos.

Exemplo: {a1, a2, a3, a4 ...}, {an}, {an}∞ n=1

Intuitivamente, dizemos que uma sequência tem limite L se os termos da sequência se aproximam de L à medida que n cresce.

Exemplo: {an}∞ = {1/n, n = 1, 2, 3, ...} tem limite 0, pois vai ficando cada vez menor e ao mesmo tempo nunca fica negativo, ou seja, nunca passa de 0.

Se esse limite existe, dizemos que a sequência converge para L e escrevemos lim an = L.
Caso contrário, dizemos que a sequência diverge.

Ainda sobre uma sequência, podemos concluir:

  • Se converge ou não
  • Se é crescente ou decrescente
  • Se existem ferramentas que possam ajudar a concluir se a sequência é ou não convergente ou divergente

Usamos a notação lim an = ∞ para dizer que a sequência diverge para ∞, indicando que à medida que n cresce, o valor de an também cresce ficando ilimitadamente enorme!

Propriedades de Sequências para {an} -> A e {bn} -> B:

  • {an} + {bn} = A + B
  • {an} - {bn} = A - B
  • {an} * {bn} = A * B
  • {an} / {bn} = A / B (desde que B seja diferente de zero)

Teorema do Confronto

Para saber se a sequência converge ou diverge, podemos usar o Teorema do Confronto, que diz que se {an} {bn} {cn} e lim an = lim cn = L, então lim bn = L.

Outro nome que damos a esse teorema é Teorema do Sanduíche.
Se o limite dos 'pães' é o mesmo, então o limite do 'recheio' também é o mesmo.
Entretanto, se os limites forem diferentes, então não podemos concluir nada.

Outro modo que podemos usar é o Teorema do Confronto para Limites, que diz que se lim an = L e lim bn = M e {an} {bn} {cn}, então L M.

Exemplo:
Se {an} = cos(n)/n e {bn} = 1/n, então -1/n cos(n)/n 1/n. Como lim -1/n = lim 1/n = 0, então lim cos(n)/n = 0 pelo teorema do confronto.

Teorema da Sequência Monótona

Também podemos usar esse teorema que diz:
se {an} é uma sequência crescente (ou decrescente... um ou o outro!) e também limitada, então ela converge.
A sequência é crescente se an+1 an para todo n e decrescente se an+1 an para todo n, e é decrescente se podemos afirmar o contrário.
E a sequência é monótona se, para valores grandes de n, ela sempre cresce ou sempre diminui (ou crescente ou decrescente).

A sequência é limitada se existe um limite superior e um limite inferior para a sequência.

Obs: as sequências são para valores de {an} para n1

Séries

Uma série é uma soma de termos de uma sequência.

Exemplo: S = a1 + a2 + a3 + a4 + ... + an no caso da série: {an}

A fórmula genérica da soma dos termos de uma sequência é

S = ((n+1)*n)/2
Exemplo:, se tenho 100 termos e quero somar todos, então S = ((100+1)*100)/2 = 5050

Se analisarmos essa formula, vemos que essa série diverge, ela explode para o infinito.

A série pode ser representada pelo símbolo Σ, que é a letra grega sigma.

Exemplo:
Dada a série {an} = {1, 2, 3, 4, 5...}
A série correspondente é S1 = 1, S2 = 1+2, S3 = 1+2+3, S4 = 1+2+3+4, S5 = 1+2+3+4+5...
Então {an} = {n} e Sn = Σn = n(n+1)/2

Entretanto, contra-intuitivamente, a soma dos termos de uma sequencia infinita pode ser finita.
Ou seja, a série (que por sua vez é uma soma de infinitos termos) pode convergir para um valor finito.

Exemplo: {an} = 1/2n = {1/2, 1/4, 1/8, 1/16...}
A série correspondente é Sn = 1/2 + 1/4 + 1/8 + 1/16 + ...
A soma dos termos tende a 1, ou seja, Sn = 1.
Usando a fórmula, Sn = (2n - 1)/2n = 1.

Dizemos que uma série converge se a sequência das somas parciais converge para um valor finito.
Nesse caso, S = lim Sn.

Sobre uma série infinita queremos saber se:

  • Converge ou não
  • Se é convergente, qual é o sua soma
  • Se existem instrumentos que nos permita tirar conclusões sobre a divergência ou convergência dela

Uma série geométrica é uma série do tipo Σa(de n=0 até ∞) rn onde a1 é o primeiro termo e r é a razão sa série.
Sendo assim, sua n-ésima soma parcial é Sn = 1 + r + r2 + r3 + ... + rn-1 = (1 - rn)/(1 - r).

Demonstração:
Considere uma série geométrica finita com o primeiro termo a e a razão r por exemplo:
S= a+ ar+ ar2+ ar3...+ arn-1
Agora vamos multiplicar a série por r:
rS = ar+ ar2+ ar3...+ arn
Agora vamos subtrair a primeira da segunda:
Rs - S= (ar+ ar2+ ar3...+ arn-1+ arn) - (a+ ar+ ar2+ ar3...+ arn-1)
então: (ar+ ar2+ ar3...+ arn-1+ arn) - (a+ ar+ ar2+ ar3...+ arn-1)
Que resulta em: rS - S = arn - a
Fatorando e simplificando temos: S(1-r) = arn - a
S = (arn - a)/(1-r)
Como o a é comum em todos os termos podemos tirar ele pra fora ficando:
S = (1-rn)/(1-r)

Teorema (série geométrica de razão r)

  • Se |r| < 1, então a série geométrica de razão r converge e sua soma será 1/(1-r)
  • Se |r| < 1 então a série geométrica diverge.
Exemplo:
Analisemos a série geométrica:
1+ 1/2 + (1/2)2... +(1/2)n
Vemos que o termo a é 1 e a razão é 1/2.
Para calcular tal soma, usamos a fórmula citada anteriormente onde:
Sn = (1-(1/2)n)/(1-1/2)
Simplificando temos: Sn = 2(1-(1/2)n).
... Agora vamos analisar.
Podemosa firmar que para qualquer valor dentro do parênteses (que está elevado a n) menor que 1, o segundo termo vai tender a zero pois vai diminuir cada vez mais.
Em contrapartida, para qualquer valor de r maior que 1, a série vai crescer infinitamente e vai divergir.
Nesse caso em específico, a série vai convergir para 2 (pois a razao é menor que 1).

Série telescopia

S = (a1 - a2) + (a2 - a3)+... (an-1 - an) + an
Essa série também é convergente pois os termos acabam se anulando.
Portanto, a soma da série telescópica é simplesmente a soma do primeiro termo e do último termo da série original.

Série harmônica

Σ1/n = 1+ 1/2+ 1/3+ 1/4 +.....

A série harmonica é divergente pois a sequencia não é limitada
Para provar isso, podemos comparar com a série geométrica 1/2n
E agora usamos o teste da comparação:
para n ≥ 1 temos 1/n ≥ 1/2n
Como a série harmonica modificada diverge para 2 e a série em questão é maior que ela, ela também deve divergir.

Teorema (Teste da divergência)

Dada a série ∑ an, se limn⭢∞ an não existir ou se existir e for diferente de zero, a série diverge.
Pois, se é um somatório, e a tenência dos numeros não é diminuir tendendo a zero, a soma nunca vai parar e vai extourar divergindo.
