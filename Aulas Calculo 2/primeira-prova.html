Sequências

Uma sequência é uma lista ordenada de termos.
Exemplo: {a1, a2, a3, a4 ...}, {an}, {an}∞ n=1
Intuitivamente, dizemos que uma sequência tem limite L se os termos da sequência se aproximam de L à medida que n cresce.
Exemplo: {an}∞ = {1/n, n = 1, 2, 3, ...} tem limite 0 pois vai ficando cada vez menor e ao mesmo tempo nunca fica negativo, ou seja, nunca passa de 0.
Se esse limite existe, dizemos que a sequência converge para L e escrevemos lim an = L.
Caso, contrário dizemos que a sequência diverge.

imagem de uma série convergente

Ainda sore uma sequência podemos dizer que:

  • Se converge ou não
  • Se é crescente ou decrescente
  • Se existem ferramentas que possam ajudar a concluir se a sequẽncia é ou não convergente ou divergente

Usamos a notação lim an = ∞ para dizer que a sequência diverge para ∞ indicando que a medida que n cresce, o valor de an também cresce ficando ilimitadamente enorme!
Propriedades de Sequências para {an} -> A e {bn} -> B:
  • {an} + {bn} = A + B
  • {an} - {bn} = A - B
  • {an} * {bn} = A * B
  • {an} / {bn} = A / B desde que B seja diferente de zero.
