Sequências

Exemplo de Sequência

Uma sequência é uma lista ordenada de termos.

Exemplo: {a1, a2, a3, a4 ...}, {an}, {an}∞ n=1

Intuitivamente, dizemos que uma sequência tem limite L se os termos da sequência se aproximam de L à medida que n cresce.

Exemplo: {an}∞ = {1/n, n = 1, 2, 3, ...} tem limite 0, pois vai ficando cada vez menor e ao mesmo tempo nunca fica negativo, ou seja, nunca passa de 0.

Se esse limite existe, dizemos que a sequência converge para L e escrevemos lim an = L.
Caso contrário, dizemos que a sequência diverge.

  • Se converge ou não
  • Se é crescente ou decrescente
  • Se existem ferramentas que possam ajudar a concluir se a sequência é ou não convergente ou divergente

Usamos a notação lim an = ∞ para dizer que a sequência diverge para ∞, indicando que à medida que n cresce, o valor de an também cresce ficando ilimitadamente enorme!

Propriedades de Sequências para {an} -> A e {bn} -> B:

  • {an} + {bn} = A + B
  • {an} - {bn} = A - B
  • {an} * {bn} = A * B
  • {an} / {bn} = A / B (desde que B seja diferente de zero)

Teorema do Confronto

Para saber se a sequência converge ou diverge, podemos usar o Teorema do Confronto, que diz que se {an} <= {bn} <= {cn} e lim an = lim cn = L, então lim bn = L.
Outro nome que damos a esse teorema é Teorema do Sanduíche.
Se o limite dos 'pães' é o mesmo, então o limite do 'recheio' também é o mesmo.
Entretanto, se os limites forem diferentes, então não podemos concluir nada.

Outro modo que podemos usar é o Teorema do Confronto para Limites, que diz que se lim an = L e lim bn = M e {an} <= {bn} <= {cn}, então L <= M.

Por exemplo:
Se {an} = cos(n)/n e {bn} = 1/n, então -1/n <= cos(n)/n <= 1/n. Como lim -1/n = lim 1/n = 0, então lim cos(n)/n = 0 pelo teorema do confronto.

Teorema da Sequência Monótona

Também podemos usar esse teorema que diz:
se {an} é uma sequência crescente (ou decrescente... um ou o outro!) e também limitada, então ela converge.
A sequência é crescente se an+1 >= an para todo n e decrescente se an+1 <= an para todo n, e é decrescente se podemos afirmar o contrário.
E a sequência é monótona se, para valores grandes de n, ela sempre cresce ou sempre diminui (ou crescente ou decrescente).

A sequência é limitada se existe um limite superior e um limite inferior para a sequência.

Obs: as sequências são para valores de {an} para n>=1

Séries

Uma série é uma soma de termos de uma sequência.

Exemplo: S = a1 + a2 + a3 + a4 + ... + an no caso da série: {an}

A fórmula genérica da soma dos termos de uma sequência é

S = ((n+1)*n)/2

Se analisarmos essa formula, vemos que essa série diverge, ela explode para o infinito.

A série pode ser representada pelo símbolo Σ, que é a letra grega sigma.

Exemplo:
Dada a série {an} = {1, 2, 3, 4, 5...}
A série correspondente é S1 = 1, S2 = 1+2, S3 = 1+2+3, S4 = 1+2+3+4, S5 = 1+2+3+4+5...
Então {an} = {n} e Sn = Σn = n(n+1)/2
