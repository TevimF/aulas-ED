Matrizes: operações e propriedades

Uma matriz é praticamente uma tabela de números. Cada número é chamado de elemento da matriz indicado por Aij. Onde i é o número da linha e j é o número da coluna.
As matrizes podem ser indentificadas como:

  • Matrizes quadradas, onde o numero de linhas é igual ao número colunas: Aij onde i=j
  • Matrizes nulas, onde todos os elementos são iguais a zero: Aij = 0
  • Matrizes diagonais, onde todos os elementos fora da diagonal principal são iguais a zero: Aij = 0 se i != j
  • Matrizes identidade, onde todos os elementos da diagonal principal são iguais a 1: Aij = 1 se i = j
  • Matrizes simétricas, onde A = AT

As operações com matrizes são:

  • Adição: A + B = C onde Cij = Aij + Bij

  • Subtração: A - B = C onde Cij = Aij - Bij

  • Multiplicação: A * B = C onde Cij = Σ Aik * Bkj

  • Multiplicação por escalar: k * A = B onde Bij = k * Aij

    • De acordo com essas operações, podemos seguir com algumas propriedades das matrizes:

    Propriedades das matrizes

    • Associativa: A + (B + C) = (A + B) + C
    • Elemento neutro: A + 0 = A
    • Elemento simétricas: A + (-A) = 0
    • Associativa: A * (B * C) = (A * B) * C
    • Distributiva: A * (B + C) = A * B + A * C
    • Comutativa: A + B = B + A
    • Elemento neutro da multiplicação: A * I = A (matriz identidade)
    • Elemento nulo da multiplicação: A * 0 = 0

    Metodo de Gauss Jordan

    O método de Gauss Jordan é um método utilizado para resolver sistemas lineares. Ele consiste em transformar a matriz em uma matriz identidade.
    Para isso, é necessário realizar operações elementares nas linhas da matriz.
    Operações elementares:

    • Troca de linhas: Trocar a posição de duas linhas da matriz
    • Multiplicação de linha por escalar: Multiplicar todos os elementos de uma linha por um escalar
    • Soma de linhas: Somar os elementos de uma linha com os elementos de outra linha

    Matriz inversa

    A matriz inversa é uma matriz que, quando multiplicada por uma matriz, resulta na matriz identidade.

    Para achar essa matriz inversa, é necessário realizar o método de Gauss Jordan com a matriz aumentada da identidade.

    Determinante

    O determinante é um número que pode ser calculado a partir de uma matriz quadrada.
    Para matrizes de tamanho 3 ou menor, podemos usar o método de Sarrus.

    Para matrizes de tamanho maior que 3, podemos usar o método de Laplace.
    Assim, escolhemos um cofator e multiplicamos pela matriz que sobrou.

    O resultado da determinante é um numero especifico.
    Se ele for zero, a matriz é linearmente dependente.
